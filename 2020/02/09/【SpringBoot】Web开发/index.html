【SpringBoot】Web开发

Posted on 2020-02-09 Edited on 2020-02-19

Web开发简介

使用Spring Boot

  • 创建SpringBoot应用，选中我们需要的模块；
  • SpringBoot已经默认将这些场景配置好了，只需要在配置文件中指定少量配置就可以运行起来；
  • 自己编写业务代码。

自动配置原理

  • xxxxAutoConfiguration：帮我们给容器中自动配置组件；
  • xxxxProperties：配置类来封装配置文件的内容。
